How to add the "Open a new tab" button to DXTabControl

This example demonstrates how to add the "Open a new tab" button to DXTabControl. Update: Starting from version 15.1, TabControl supports this functionality out of the box. Please refer to this article for more details: Adding and Removing Tab Items (https://documentation.devexpress.com/#WPF/CustomDocument113904) (update the example description).
